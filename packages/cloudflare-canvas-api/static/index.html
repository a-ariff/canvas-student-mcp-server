üê≥ Docker Deployment (Local)

Run locally with Docker Compose for stdio-based MCP 

# Clone repository
git clone https://github.com/a-ariff/canvas-student-mcp-server.git
cd canvas-student-mcp-server/packages/canvas-student-mcp-server

# Create .env file
cat > .env << EOF
CANVAS_API_KEY=your_canvas_api_token
CANVAS_BASE_URL=https://canvas.instructure.com
DEBUG=false
EOF

# Start with Docker Compose
docker-compose up -d

# Interact via stdio
docker exec -it canvas-student-mcp-server /bin/sh

ü§ñ Custom MCP Client

Build your own MCP client using the SDK 

# Install MCP SDK
npm install @modelcontextprotocol/sdk

# Connect to server
import { Client } from '@modelcontextprotocol/sdk/client/index.js';
import { SSEClientTransport } from '@modelcontextprotocol/sdk/client/sse.js';

const transport = new SSEClientTransport(
  new URL('https://canvas-mcp-sse.ariff.dev/sse')
);

const client = new Client({
  name: 'my-mcp-client',
  version: '1.0.0'
}, { capabilities: {} });

await client.connect(transport);
