GET /courses/{userId}

Get assignments for a specific course

Get user profile information

Health check endpoint for monitoring

üíª Example Usage

Once you have your user ID, you can make requests like this:

JavaScript (Fetch API)

// Get your courses
const response = await fetch('https://canvas-api.yourdomain.com/courses/YOUR_USER_ID');
const courses = await response.json();

// Get assignments for a course
const assignments = await fetch('https://canvas-api.yourdomain.com/assignments/YOUR_USER_ID?course_id=123');

// Custom Canvas API request
const customRequest = await fetch('https://canvas-api.yourdomain.com/canvas/YOUR_USER_ID', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    endpoint: '/courses/123/modules',
    method: 'GET'
  })
});

Python (requests)

import requests

# Get your courses
response = requests.get('https://canvas-api.yourdomain.com/courses/YOUR_USER_ID')
courses = response.json()

# Custom Canvas API request
custom_response = requests.post('https://canvas-api.yourdomain.com/canvas/YOUR_USER_ID',
    json={
        'endpoint': '/courses/123/assignments',
        'method': 'GET'
    }
)

curl

# Get your courses
curl https://canvas-api.yourdomain.com/courses/YOUR_USER_ID

# Custom Canvas API request
curl -X POST https://canvas-api.yourdomain.com/canvas/YOUR_USER_ID \
  -H "Content-Type: application/json" \
  -d '{"endpoint": "/courses", "method": "GET"}'

  }
