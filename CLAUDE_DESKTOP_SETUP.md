# Claude Desktop Integration Setup\n\nThis guide explains how to set up the Canvas Student MCP Server with Claude Desktop.\n\n## Prerequisites\n\n1. **Claude Desktop**: Download and install [Claude Desktop](https://claude.ai/desktop)\n2. **Python 3.8+**: Ensure Python is installed with `python3 --version`\n3. **Canvas Account**: Active student account for your Canvas LMS\n4. **Chrome/Chromium**: Required for Selenium web automation\n\n## Quick Setup\n\n### 1. Install Dependencies\n\n```bash\ncd canvas-student-mcp-server\nnpm run install:all\n```\n\n### 2. Configure Environment\n\n```bash\n# Copy the example environment file\ncp packages/canvas-student-mcp-server/.env.example packages/canvas-student-mcp-server/.env\n\n# Edit the .env file with your Canvas credentials\n# IMPORTANT: Use YOUR actual Canvas URL and credentials\n```\n\n**Edit `.env` file:**\n```bash\nCANVAS_BASE_URL=https://your-school.instructure.com\nCANVAS_USERNAME=your.email@school.edu\nCANVAS_PASSWORD=your_password\n```\n\n### 3. Configure Claude Desktop\n\n**Option A: Automatic Configuration**\n```bash\n# Copy the provided configuration\ncp claude-desktop-config.json ~/.config/claude-desktop/config.json\n```\n\n**Option B: Manual Configuration**\n\nEdit Claude Desktop's configuration file:\n- **macOS**: `~/Library/Application Support/Claude/claude_desktop_config.json`\n- **Windows**: `%APPDATA%\\Claude\\claude_desktop_config.json`\n- **Linux**: `~/.config/claude-desktop/config.json`\n\nAdd this configuration:\n\n```json\n{\n  \"mcpServers\": {\n    \"canvas-student-mcp\": {\n      \"command\": \"python3\",\n      \"args\": [\n        \"PATH_TO_THIS_REPO/packages/canvas-student-mcp-server/app.py\"\n      ],\n      \"env\": {\n        \"PYTHONUNBUFFERED\": \"1\",\n        \"PYTHONPATH\": \"PATH_TO_THIS_REPO/packages/canvas-student-mcp-server/src\"\n      }\n    }\n  }\n}\n```\n\n**Replace `PATH_TO_THIS_REPO` with the actual path to this repository!**\n\n### 4. Restart Claude Desktop\n\nRestart Claude Desktop completely for the configuration to take effect.\n\n## Usage\n\nOnce configured, you can use these commands in Claude Desktop:\n\n### Authentication\n```\nPlease authenticate with Canvas using my credentials.\n```\n\n### Get Courses\n```\nShow me all my Canvas courses.\n```\n\n### Course Content\n```\nGet all modules for course ID 12345.\nShow me assignments for my Technology Management course.\nSearch for \"assignment\" in course 12345.\n```\n\n### Build Course Index\n```\nBuild a searchable corpus for course 12345.\n```\n\n## Available Tools\n\n1. **authenticate_student** - Authenticate with Canvas credentials\n2. **get_student_courses** - List all enrolled courses\n3. **get_course_modules** - Get modules for a specific course\n4. **get_module_content** - Get content within a module\n5. **get_course_assignments** - Get assignments for a course\n6. **search_course_content** - Search through course content\n7. **build_course_corpus** - Index course content for searching\n\n## Troubleshooting\n\n### MCP Server Not Found\n- Verify the path in your Claude Desktop config\n- Ensure Python dependencies are installed\n- Check that `python3` is in your system PATH\n\n### Authentication Fails\n- Verify Canvas URL is correct\n- Check username/password in .env file\n- Ensure Canvas allows programmatic access\n- Try logging in manually to Canvas first\n\n### Chrome/Selenium Issues\n```bash\n# Install Chrome/Chromium\n# Ubuntu/Debian:\nsudo apt-get install chromium-browser\n\n# macOS:\nbrew install --cask google-chrome\n\n# Windows: Download from Google Chrome website\n```\n\n### Permission Errors\n```bash\n# Ensure proper file permissions\nchmod +x packages/canvas-student-mcp-server/app.py\n```\n\n## Security Notes\n\n- **Credentials stay local**: Your Canvas credentials never leave your machine\n- **Environment variables**: Use `.env` file, never commit credentials\n- **Compliance**: Ensure your usage complies with your institution's policies\n- **Rate limiting**: The server respects Canvas rate limits\n\n## Testing the Setup\n\nTest if everything works:\n\n```bash\n# Test the MCP server directly\ncd packages/canvas-student-mcp-server\npython3 app.py\n```\n\nIf the server starts without errors, the setup is correct.\n\n## Need Help?\n\nCheck the troubleshooting guides:\n- `packages/canvas-student-mcp-server/TROUBLESHOOTING.md`\n- `packages/canvas-student-mcp-server/MCP_INTEGRATION.md`\n\nOr open an issue on GitHub: https://github.com/a-ariff/canvas-student-mcp-server/issues\n"
